Классификация


I. Международная классификация болезней X пересмотра (МКБ-X):


Шифр болезни по МКБ-10:

  • A84.9 – Клещевой вирусный энцефалит неуточненный


Общепринятой классификации клещевого вирусного энцефалита нет.


II. Рекомендуемая клиническая классификация клещевого вирусного энцефалита:


По типу (клинической форме):

I. Типичные формы:

  • неочаговые:

1. лихорадочная

2. менингеальная

  • очаговые:

1. менингоэнцефалитическая

2. полиоэнцефалитическая

3. полиоэнцефаломиелитическая

4. полиомиелитическая

5. двухволновое течение (с указанием формы второй волны).


II. Атипичные формы:

1. инаппарантная (бессимптомная)


По длительности течения:

острое течение

хроническое течение


По тяжести течения:

1. легкой степени тяжести

2. средней степени тяжести

3. тяжелой степени тяжести


По характеру течения:

1. гладкое (неосложненное)

2. негладкое (осложненное)


Классификация хронических форм клещевого вирусного энцефалита

(К.Г.Уманский, В.Н.Коваленко, А.В.Субботина, Е.П.Деконенко, 1984 г.)


1. Клинические формы

1.1 Гиперкинетическая (синдромы: эпилепсия Кожевникова, миоклонус-эпилепсия, гиперкинетический)

1.2 Амиотрофическая (синдромы : полиомиелитический, энцефалополиомиелитический, рассеянного энцефаломиелита, бокового амиотрофического склероза)

1.3 Редко встречающиеся синдромы, не относящиеся к формам 1.1 и 1.2


2. Степень тяжести синдрома

2.1 Лёгкая (трудоспособность сохранена)

2.2 Средняя (инвалидность 3 группы)

2.3 Тяжелая (инвалидность 1 и 2 групп)


3. По времени возникновения хронического процесса

3.1 Инициальный прогредиентный (непосредственное продолжение острого КЭ)

3.2 Ранний прогредиентный (первично-хроническая; возникает в течение первого года после острого КЭ)

3.3 Поздний прогредиентный (вторично-хроническая; возникает спустя год и более после острого КЭ)

3.4 Спонтанный прогредиентный (возникает без отчетливого острого КЭ)


4. По характеру течения хронического КЭ

4.1 Рецидивирующий

4.2 Непрерывно прогрессирующий

4.3 Абортивный


5. Стадии заболевания

5.1 Начальная

5.2 Нарастание (прогрессирование)

5.3 Стабилизация

5.4 Терминальная
