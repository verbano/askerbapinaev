Классификация


Шифр болезни по МКБ-10:

  • В06.


I. Международная классификация болезней X пересмотра (МКБ-X):


В06 – Краснуха (немецкая корь)

В06.0 – Краснуха с неврологическими осложнениями

  • энцефалит (G05.1)
  • менингит (G02.0)
  • менингоэнцефалит (G05.1)

В06.8 – Краснуха с другими осложнениями

  • артрит (M01.4)
  • пневмония (J17.1)

В06.9 – Краснуха без осложнений


Общепринятой классификации клинических форм краснухи нет.

А.П.Казанцев (1970 г.) выделял следующие клинические формы болезни:


Приобретенная краснуха:

1. Типичная форма различных степеней тяжести (легкая, средняя, тяжелая степени тяжести), неосложненное, или осложненное течение (артрит, энцефалит, тромбоцитопеническая пурпура и др.)

2. Атипичная форма (без сыпи).

3. Стертая (субклиническая).

4. Инаппарантная (бессимптомная)


Врожденная краснуха:

1. Поражение нервной системы.

2. Врожденные пороки сердца.

3. Форма с поражением слуха.

4. Форма с поражением глаз.

5. Смешанные формы.

6. Резидуальные явления врожденной краснухи.


II. Рекомендуемая клиническая классификация краснухи:


По типу (клинической форме):

1. Типичная (манифестная)

2. Атипичная:

2.1 с изолированным синдромом экзантемы

2.2 с изолированным синдромом лимфаденопатии

2.3 стертая (субклиническая)

2.4 инаппарантная (бессимптомная)


По тяжести течения:

1. легкой степени тяжести

2. средней степени тяжести

3. тяжелой степени тяжести


По характеру течению (наличию осложнений):

1. гладкое (неосложненное)

2. негладкое (осложненное)
